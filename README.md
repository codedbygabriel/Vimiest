# Vimiest

A Web To do App that uses vim motions for some reason.

## Usage

All data is stored in `LocalStorage` in the `_vimiest_todos` Key. As Long as the LocalStorage is not cleaned, your data will be safe.

[Open App](https://codedbygabriel.github.io/Vimiest/)

## Keybinds

| Keys | Action |
-------|---------
| `<f>-a` | Opens Add Menu |
| `<f>-r` | Enables Remove Mode |
| `<f>-c` | Clears All To Dos |
| `<f>-h` | Opens CheatSheet (WIP) |
| `<f>` | Outlines Buttons -a-r-c |
